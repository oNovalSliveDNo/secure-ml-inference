# Датасеты проекта

## Breast Cancer Wisconsin Diagnostic

### Источник и описание

Датасет встроен в библиотеку scikit-learn: `sklearn.datasets.load_breast_cancer`.
Он содержит характеристики клеточных ядер, полученные из изображений тонкоигольной аспирационной биопсии (FNA) опухолей молочной железы.

### Характеристики

- Количество объектов: 569.
- Количество признаков: 30 числовых признаков.
- Задача: бинарная классификация.
- Целевая переменная: `malignant` / `benign`.
- Пропуски отсутствуют.
- Предоставляется также DataFrame с названиями признаков.

### Использование в проекте

- Пайплайн модели: `StandardScaler + LogisticRegression`.
- Используется для защищённого classification inference.
- При защищённом инференсе клиент применяет тот же scaler перед шифрованием.
- Веса модели сохраняются и используются сервером в целочисленном виде.

## Diabetes

### Источник и описание

Датасет встроен в библиотеку scikit-learn: `sklearn.datasets.load_diabetes`.
Он содержит числовые медицинские признаки пациентов, используемые для оценки прогрессирования диабета.

### Характеристики

- Количество объектов: 442.
- Количество признаков: 10 числовых признаков.
- Задача: регрессия.
- Целевая переменная: количественный показатель прогрессирования диабета.
- Пропуски отсутствуют.
- Предоставляется также DataFrame с названиями признаков.

### Использование в проекте

- Пайплайн модели: `StandardScaler + Ridge`.
- Используется для защищённого regression inference.
- При защищённом инференсе клиент применяет тот же scaler перед шифрованием.
- Веса модели сохраняются и используются сервером в целочисленном виде.
